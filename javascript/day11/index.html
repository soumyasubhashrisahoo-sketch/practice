what is single threaded multi-threaded

  • Single-threaded: A single thread of execution is used to run the program. Only one task can be performed at a time.
  • Multi-threaded: Multiple threads of execution are used to run the program. Multiple tasks can be performed simultaneously.

what is synchronous and asynchronous

  • Synchronous: Tasks are performed one after the other. Each task must wait for the previous task to complete before it can start.
  • Asynchronous: Tasks can be performed independently of each other. A task can start before the previous task has completed, allowing for more efficient use of resources and improved performance.

QUESTION 1-WHY WE NEED ASCHYNOROUS TYPE OF CODING

  • To avoid blocking the execution of other code while waiting for a long-running task to complete.
  • So our apps and websites don't freeze while waiting for slow tasks to finish.
  • To improve the performance and responsiveness of applications by allowing multiple tasks to run concurrently., Asynchronous coding is essential to keep applications responsive and efficient by allowing long-running tasks—such as API requests, database queries, or file I/O—to run in the background without freezing the main thread or user interface

WHAT IS CALL BACK FUNCTION - AND HOW WE CAN USE IT TO GRAB CALL

A callback function is a function that is passed as an argument to another function and is executed after the completion of that function. It is a way to handle asynchronous operations in JavaScript.
