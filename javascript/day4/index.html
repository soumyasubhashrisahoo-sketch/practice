DOM - Document Object Model

what is DOM?

a programming interface that represents a web page's structure (HTML/XML) as a

tree

of objects, allowing scripts like JavaScript to

dynamically access, change, add, or delete content, structure, and style

, making websites interactive without reloading. When a browser loads a page, it builds this tree, treating each HTML tag as a node, which scripts can then traverse and manipulate to create dynamic features like animations, form validation, and real-time updates dom is a heavy object - it takes time to load the dom object in the memory

why dom is like tree like structure ?

because it maps the hierarchical, nested nature of HTML code into a parent-child node structure.

difference between dom and html

DOM Traversal

DOM traversal refers to the process of navigating through the nodes of a DOM tree. It allows developers to move from one element to another, such as from a parent element to its child elements, or from a child element back to its parent. This is essential for accessing and manipulating specific elements in the DOM.
